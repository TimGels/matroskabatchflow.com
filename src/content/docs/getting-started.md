## Prerequisites

Before using Matroska Batch Flow, make sure you have the following:

| Requirement    | Notes                                       |
| -------------- | ------------------------------------------- |
| **Windows 11** | Currently only Windows builds are available |

The Windows (WinAppSDK) release includes all dependencies out of the box, including `mkvpropedit` from MKVToolNix. No additional downloads are required.

:::tip
If you prefer to use your own installation of mkvpropedit, you can configure its path in **Settings**.
:::

## Download

Head to the [GitHub Releases](https://github.com/TimGels/Matroska-Batch-Flow/releases) page and download the latest release for your platform.

### Installation Steps

1. Download the release archive (`.zip`) from GitHub Releases
2. Extract the archive to a folder of your choice
3. Run `MatroskaBatchFlow.exe`

## Next Steps

Once Matroska Batch Flow is running, head over to the [User Guide](/docs/user-guide) to learn the basic workflow.
