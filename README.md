# matroskabatchflow.com

Website and documentation for [Matroska Batch Flow](https://github.com/TimGels/Matroska-Batch-Flow).

Deployed at [matroskabatchflow.com](https://matroskabatchflow.com/).

## Development

```bash
npm install
npm run dev
```

The dev server runs at `http://localhost:4321`.

## Build

```bash
npm run build
```

Static output is generated in the `dist/` directory.

## Deployment

Pushes to `main` trigger a GitHub Actions workflow that builds the site and deploys it to GitHub Pages.
